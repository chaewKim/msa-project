📌 MSA Project – 물류 시스템 예제
✅ 프로젝트 소개
Spring Boot 기반 MSA(마이크로서비스 아키텍처) 실습 프로젝트입니다.
상품, 주문, 인증 서비스를 분리하고 Eureka, Gateway, Redis, Zipkin 등을 활용하여 실제 MSA 환경을 구성합니다.

기술	사용 내용
Java 17	개발 언어
Spring Boot 3.x	서비스 개발
Spring Cloud	MSA 구성
PostgreSQL	데이터베이스
Redis	캐싱 서버
Docker	컨테이너화
Docker-Compose	통합 실행 환경
Zipkin	분산 추적
Eureka	서비스 디스커버리
Spring Cloud Gateway	API Gateway
Feign Client	서비스 간 통신
